# Welcome to Skript Parser

Skript Parser is a **standalone Java implementation** of the Skript language, designed to be cleaner, more flexible, and easier to understand than the original Skript plugin. It provides a complete parsing and execution framework that can be embedded in any Java application.

## What is Skript?

Skript is a language originally created for Minecraft servers that makes scripting accessible to everyone. Even without programming experience, Skript code is highly readable:

```vba
on right click on dirt:
    if the player is holding seeds:
        set the clicked block to grass
    else if the player is holding any mushroom:
        set the clicked block to mycelium
```

This natural syntax makes it perfect for rapid prototyping and accessible scripting.

## Why Skript Parser?

The original Skript plugin had some limitations:
- Tightly coupled with the Spigot/Minecraft API
- Difficult to maintain and extend
- Only usable within Minecraft servers
- Complex, tangled codebase

Skript Parser addresses these issues by providing:

## Project Goals

The main aim of this project is to provide a **better, easier to understand, and more flexible implementation** of the Skript language. This means:

- **Standalone library**: Can be used in any Java environment
- **Improved API**: Cleaner interfaces for registering syntax and handling events
- **Better documentation**: Comprehensive guides and API references
- **Modern practices**: Uses current Java features and best practices
